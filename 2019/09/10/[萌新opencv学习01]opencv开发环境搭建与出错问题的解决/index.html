[萌新opencv学习01]opencv开发环境搭建与出错问题的解决

首先我用VS2017的结果被人叫我去用2015开发，我去把2017卸载了。（千万别！17也能开发的）

然后走一个大佬给的教程Cmake来设置环境 结果 一直各种错误

Selecting Windows SDK version  to target Windows 10.0.17763.
CMake Error at CMakeLists.txt:127 (project):
  Failed to run MSBuild command:

    C:/Program Files (x86)/MSBuild/14.0/bin/MSBuild.exe

然后我去问一个大佬他说你干嘛卸载 你现在环境已经混乱了 重装吧，woc！！后悔死

当然我没重装，我手动添加环境变量了。。。。（VS2017 opencv3.42）
下面开始说我安装之路和各种坑！：（不是cmake）
我是看这个视频的 https://www.bilibili.com/video/av29600072/?p=1
先别看！！听我吐槽完视频的坑
首先老师讲的蛮不错的，当时有几个坑 第一他的版本是opencv3.10 在添加依赖项的时候他填了 opencv_world310d.lib （坑！！！！）解读一下 你安装什么版本你要把310改成你的版本号 我是opencv_world342d.lib。巨坑 如果搞错了更改超级麻烦（先去出厂C++的环境）再按照视频添加依赖项（VC++目录不会被出厂，所以不用再次弄了）

第二个坑、

你从电脑复制路径都是像上图一样的 \ 斜杠 可是在C++会找不到路径的
你要把 \ 改成 / 就可以了

搞定了 然后祝你好运

贴上代码

#include<opencv2/opencv.hpp>
#include<iostream>

using namespace cv;
int main(int argc, char** argv) {
    Mat src = imread("D:/Temp/181204202052.jpg");
    if (src. empty()) {
        printf("could not load image...\n");
        return -1;
    }
    namedWindow("test opencv setup", CV_WINDOW_AUTOSIZE);
    imshow("test opencv setup", src);

    waitKey(0);
    return 0;
}
# opencv
